### Speciedex Website: ("https://speciedex.org") - A website for the Speciedex Project. Mirrored on Bitcoin.org's website and design.
# Speciedex.org

Speciedex is a decentralized network leveraging Bitcoin and the Lightning Network as foundational layers. This placeholder site mirrors the clean design of Bitcoin.org, integrating resources and links to related projects.


## Setup
1. Clone the repository:
```
git clone https://github.com/YOUR-USERNAME/speciedex.org.git
```

2. Serve the site locally using Python or Node.js:
```
python3 -m http.server
```

or

```
npx serve
```

3. Access the site at `http://localhost:8000`
